Install Theme

When purchasing Dentistry theme from Themeforest, you will be able to download Dentistry files with your Themeforest account. The Main Files contains all files, you have to select Installable WordPress Theme files with just necessary installable WordPress theme files.

Install dentistry theme via wordpress.

  1. Unzip the downloaded package from Themeforest. Find the folder theme and inside it find the file dentistry.zip
  2. Login to your WordPress dashboard, go to Appearance > Themes  > click Install Themes tab and select Upload
  3. Click the Browse button and find the file"dentistry.zip" on your computer.Click on "Install"
  4. Wait the theme being uploaded and installed.
  5. Activate the theme: Go to Appearance > Themes and Activate.

Install dentistry theme via FTP

Follow the steps below to install dentistry via FTP. 

  1. Log into your hosting space via an FTP software
  2. Unzip the dentistry.zip file and ONLY use the extracted dentistry theme folder
  3. Upload the extracted dentistry theme folder into wp-content > themes folder
  4. To activate the theme: Go to Appearance > Themes select dentistry theme and activate

Common Install Error: Are You Sure You Want To Do This?

The “Are You Sure You Want To Do This” message may occur, means that you are limited in upload file size. Consider install the theme via FTP if this happens, or call your hosting company and ask them to increase the limited size.

Menu Settings

  1. Navigate to Appearance > Menus section of your admin.
  2. Click the "Create A New Menu" to create new menu.
  3. To add a menu item, select one of your created pages on the left hand side and click "Add to Menu". You can also add all kinds of different posts from the left side.
